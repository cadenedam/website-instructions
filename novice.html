Novice: Publish your first website

Home | Expert

Step 1 — Create a GitHub account

If you don't have one, go to github.com and sign up.

Step 2 — Create the repository

  1. Click "New repository" and name it USERNAME.github.io (replace USERNAME).
  2. Make the repo Public and create it.

Why this name? GitHub serves this repo as your personal site at https://USERNAME.github.io.

Step 3 — Create files locally

Create index.html, novice.html, expert.html and a css/style.css file. (You can copy these from this repository.)

Step 4 — Push to GitHub

git init
git branch -M main
git remote add origin https://github.com/USERNAME/USERNAME.github.io.git
git add .
git commit -m "Initial site"
git push -u origin main

Step 5 — Enable Pages

Open the repository on GitHub → Settings → Pages and choose the branch main and folder / (root), then Save.

Step 6 — Add screenshots

Drop images into assets/, then reference them with <img src="assets/example.png">.
